# 🎨 UI/UX Task 2 – High-Fidelity UI Design

## 📌 Overview
This project is part of my UI/UX Internship at CodeAlpha.

The objective of this task is to convert low-fidelity wireframes into a high-fidelity UI design with colors, images, typography, and modern UI elements.

---

## 🎯 Objective
- Create visually appealing UI design  
- Apply color theory and typography  
- Improve usability and aesthetics  

---

## 📱 Project Description
I designed a high-fidelity UI for a Food Delivery mobile application.

The design includes colors, icons, images, and proper layout alignment to create a modern and user-friendly interface.
